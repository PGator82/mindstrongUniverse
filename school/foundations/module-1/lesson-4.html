Lesson 4: Compare Numbers Without Guessing

Goal: Compare multi-digit numbers using place value, not vibes.

Hook

Comparing numbers should feel like reading a scoreboard. If it feels like a puzzle, you’re missing the method.

The Hack

Compare from the left. The first place that’s different decides the winner.

  • Thousands decide before hundreds.
  • Hundreds decide before tens.
  • Tens decide before ones.

Standard Method

Align numbers by place value and compare digits from greatest place to least.

Guided Practice

  1. Which is greater: 4,205 or 4,052? Explain in one sentence.
  2. Put these in order least → greatest: 900, 1,090, 1,009.
  3. True/False: 3,999 is greater than 4,001.
  4. Choose the correct symbol: 6,120 __ 6,102.

Boss Problem

A student says 5,099 is bigger than 5,900 because “99 is bigger than 900.” Fix their thinking using the left-to-right method.

Exit Check

Which is greater: 8,407 or 8,470?

Answer

8,470 (compare hundreds: 4 = 4, then tens: 7 > 0)
