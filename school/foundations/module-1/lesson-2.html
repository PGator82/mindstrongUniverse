Lesson 2: Place Value Is a Volume Knob

Goal: Understand how digit position changes value by powers of 10.

Hook

Moving a digit one place changes everything. It’s not decoration — it’s power.

The Hack

Each place to the left multiplies by 10. Each place to the right divides by 10.

Same digit. Different volume: 5, 50, 500.

Standard Method

Digits have different values depending on place: ones, tens, hundreds, thousands, etc.

Guided Practice

  1. What is the value of the 7 in: 17, 70, 700?
  2. Write the value of the underlined digit in 3,482 (the 4).
  3. Which number is “louder”: 406 or 460? Explain.

Boss Problem

A student says: “The 6 in 64 and the 6 in 604 are the same.” Are they right? Explain.

Exit Check

Moving one place to the left multiplies value by ____.

Answer

10
