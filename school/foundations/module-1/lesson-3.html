Lesson 3: Expanded Form — Breaking Numbers Apart

Goal: Break numbers into place-value pieces to understand structure.

Hook

Numbers aren’t single things. They’re built from parts. Break them apart = control them.

The Hack

Expanded form is number LEGO.

3,402 = 3,000 + 400 + 2

Standard Method

Expanded form shows each digit’s value as a sum of its place values.

Guided Practice

  1. Write 5,216 in expanded form.
  2. Write 902 in expanded form.
  3. Which part of 7,430 is missing (think thousands/hundreds/tens/ones)?

Boss Problem

A student writes: 4,080 = 4,8. Explain why this is wrong using expanded form.

Exit Check

Write 1,205 in expanded form.

Answer

1,000 + 200 + 5
